# Quick Start Guide

## Testing Locally

To test your website before uploading:

### Windows
1. Navigate to the AawaraWeb folder
2. Right-click on `index.html`
3. Select "Open with" → Choose your browser
4. The website will load locally

Alternatively:
- Double-click `index.html` to open in default browser

### Mac/Linux
1. Open terminal in the AawaraWeb folder
2. Run: `python3 -m http.server 8000`
3. Visit: `http://localhost:8000` in your browser

## Performance Tips

- Optimize image file sizes before uploading
- Use PNG for icons, JPG for photos
- Ensure images are appropriately sized
- Website loads instantly (no dependencies)
- Works offline (fully static)

## Troubleshooting

**Images not showing?**
- Check filenames match exactly (case-sensitive)
- Ensure images are in the assets folder
- Verify file formats (.png or .jpg)

**Layout broken on mobile?**
- This shouldn't happen - CSS is responsive
- Clear browser cache and reload
- Try a different browser

**Links not working?**
- Navigation links should work between index.html and privacy-policy.html
- Update social media links with actual URLs
- Ensure external links have full URL (https://...)
