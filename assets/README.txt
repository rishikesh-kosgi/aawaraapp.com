# Assets Folder

Add your images here with these filenames:

## Required Images

1. **hero-image.png** (500x600px recommended)
   - Main hero section image
   - Should represent your app or its main feature

2. **showcase-image.png** (450x500px recommended)
   - Showcase section image
   - Complements the app features description

3. **feature-1.png** to **feature-4.png** (80x80px recommended)
   - Feature icons for the 4 main features
   - Innovation, Security, Performance, Support icons

## Image Format Guidelines

- Use PNG for icons and graphics
- Use JPG for photographs
- Ensure images are optimized (not too large)
- High resolution for clarity on all devices
- Brand-consistent colors

## Temporary Placeholder

Until you add your images, the website will appear with broken image links. This is expected. Once you add images with the correct filenames, they'll display automatically.

## Tools for Creating Images

If you need placeholder images quickly:
- Unsplash, Pexels, or Pixabay for free stock photos
- Figma or Canva for creating icons
- Adobe XD or Sketch for design elements
